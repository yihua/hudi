vllm {{ .Chart.Version }} deployed, serving {{ .Values.model }}.

First boot downloads the model weights -- expect several minutes before the
pod is Ready (watch: kubectl -n {{ .Release.Namespace }} get pods -w).

Point the hudi-agent-gateway at it:

  helm upgrade hudi-agent-gateway <chart> --reuse-values \
    --set gateway.provider=openai-compatible \
    --set gateway.openaiCompatible.baseUrl=http://{{ include "vllm.fullname" . }}.{{ .Release.Namespace }}.svc:{{ .Values.service.port }}/v1 \
    --set gateway.model={{ .Values.model }}

Sanity check:
  kubectl -n {{ .Release.Namespace }} port-forward svc/{{ include "vllm.fullname" . }} 8001:{{ .Values.service.port }}
  curl localhost:8001/v1/models
